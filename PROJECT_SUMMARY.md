## 🚀 启动方式

### 方式1：Mock模式（推荐，无需数据库）

#### Windows：
```bash
双击运行: start-mock.bat
```

#### Linux/Mac：
```bash
chmod +x start-mock.sh && ./start-mock.sh
```

**包含功能：**
- ✅ Mock API服务器（2种微量元素、3个食谱、3种体质）
- ✅ 完整用户认证
- ✅ 所有功能可测试

### 方式2：真实数据库模式

1. 安装PostgreSQL
2. 配置 `backend/.env`
3. 运行 `start.bat` (Windows) 或 `start.sh` (Linux/Mac)
