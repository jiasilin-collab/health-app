## 🌐 访问Web应用

### 本地访问（推荐）

```
http://localhost:3000
```

**在电脑浏览器中打开上述地址即可访问应用！**

### 手机访问

#### 获取电脑IP地址

**Windows:**
```bash
# 方法1：使用ipconfig
ipconfig

# 方法2：使用PowerShell
Get-NetIPAddress -AddressFamily IPv4 | Select-Object IPAddress

# 方法3：使用设置
设置 > 网络和Internet > 查看网络属性
```

**Mac/Linux:**
```bash
# 使用ifconfig
ifconfig

# 或使用ip addr
ip addr show
```

**常见IP格式：**
- 192.168.1.x
- 10.0.0.x
- 172.16.x.x

#### 在手机浏览器访问

```
http://[你的电脑IP]:3000
```

**例如：**
```
http://192.168.1.100:3000
http://10.0.0.5:3000
```

## ⚠️ 常见问题

### Q: 页面无法访问？

**A:**
1. 确认Mock API服务器正在运行
2. 检查端口3000是否被占用
3. 尝试使用http://127.0.0.1:3000

### Q: API请求失败？

**A:**
1. 打开浏览器开发者工具
2. 查看Network标签中的请求
3. 确认请求URL正确
4. 查看响应状态码和错误信息

### Q: 手机无法访问电脑IP？

**A:**
1. 确认手机和电脑在同一WiFi
2. 检查电脑防火墙设置
3. 检查手机网络连接
4. 尝试使用电脑的局域网IP而非公网IP

### Q: 页面样式错乱？

**A:**
1. 清除浏览器缓存
2. 硬刷新页面（Ctrl+F5）
3. 检查浏览器控制台是否有CSS错误
4. 尝试使用不同浏览器
